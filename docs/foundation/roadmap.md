# Roadmap

ETHDS is built in phases. Each phase has a full scope document under
[`docs/phases/`](../phases/) — this page is just a status index. Do not
duplicate phase content here; update the phase file and this status
column instead.

A phase does not start until the phases it depends on (listed in each
phase file) are complete.

## Status Legend

- ✅ **Complete** — deliverables built, merged, and verified
- 🚧 **In Progress** — actively being built
- 📋 **Planned** — scope documented, not yet started

## v1.0 Readiness

ETHDS reaches v1.0 when every phase above is complete and the
[Phase 12 audit](../phases/phase-12-final-release.md) confirms the
criteria in [`success-metrics.md`](success-metrics.md) are met.
