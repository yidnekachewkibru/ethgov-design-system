# Phase 4 — Documentation Platform

**Status:** 📋 Planned

## Goal

Build the public ETHDS documentation site so design principles, brand
system, components, patterns, and contribution guidance are discoverable
and usable by government teams and contributors.

## Deliverables

- Docusaurus Setup
- Site Architecture
- Navigation
- Search
- Multilingual Documentation (the docs site itself must support all six
  ETHDS languages, per [Phase 6](phase-6-localization-framework.md))
- Accessibility Documentation (the site itself must meet WCAG 2.2 AA, and
  must document the project's accessibility standards)
- Deployment Pipeline
- GitHub Pages Deployment

## Output Location

`packages/ethds-docs/` (replaces the README-only stub at
[`packages/ethds-docs/README.md`](../../packages/ethds-docs/README.md)).

## Dependencies

- [Phase 1 — Design Principles](phase-1-design-principles.md) through
  [Phase 3 — Design Tokens](phase-3-design-tokens.md) for initial content
  to publish. Content from later phases (5–9) is added incrementally as
  those phases complete.
