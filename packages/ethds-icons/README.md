# @ethds/icons

**Status:** Planned. A starter set (Check, Info, Warning, Error, Search,
chevrons) ships inside [`@ethds/react`](../ethds-react/); this package is
the future home of the full government icon set.

## Purpose

The ETHDS icon set: a consistent, accessible (`aria-hidden` by default,
with labeled-icon variants), optimized SVG icon library covering
government service iconography (documents, payments, identity,
notifications, navigation, status).

## Depends On

  iconography guidelines (stroke weight, grid, color usage)
