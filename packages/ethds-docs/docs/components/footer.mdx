import { Footer } from '@ethds/react';

<Footer

/>

## When to use this component

On every page. Standard homes for the accessibility statement, privacy,
contact, and cross-government links.

## When not to use this component

Don't bury primary tasks in the footer — it's for secondary links.

## Accessibility

- A `<footer>` contentinfo landmark; each link group is its own labelled
  `<nav>` with a heading, so screen-reader users can navigate the footer
  by section.
- Link colours on the dark surface meet AA; focus rings are visible.

## Props

| Prop | Type | Notes |
|---|---|---|
| `groups` | `{ title, links: { label, href }[] }[]` | grouped link navs |
| `copyright` | `ReactNode` | legal line |
| `label` | `string` | optional landmark name |
