# Breadcrumb

Shows the citizen where they are in the site hierarchy and provides a way
back up.

<Example>
  <Breadcrumb
    items={[
      { label: 'Home', href: '#' },
      { label: 'Services', href: '#' },
      { label: 'Renew licence' },
    ]}
  />
</Example>

```tsx
import { Breadcrumb } from '@ethds/react';

<Breadcrumb
  items={[
    { label: 'Home', href: '/' },
    { label: 'Services', href: '/services' },
    { label: 'Renew licence' },          // current page — no href
  ]}
/>
```

## When to use this component

Use on pages more than one level deep — ministry/agency pages, service
pages — so citizens can orient and move up a level.

## When not to use this component

Don't use on the homepage, or inside a multi-step application flow (the
flow's own Back/step navigation does that job).

## Accessibility

- A `<nav>` landmark with an accessible name (pass `label` in the active
  language) containing an ordered list.
- The last item is the current page: rendered as text (not a link) with
  `aria-current="page"`.
- Separators are decorative icons hidden from assistive tech.

## Plain HTML

```html
<nav aria-label="Breadcrumb" class="ethds-breadcrumb">
  <ol class="ethds-breadcrumb__list">
    <li class="ethds-breadcrumb__item">
      <a href="/" class="ethds-breadcrumb__link">Home</a>
      <span class="ethds-breadcrumb__separator" aria-hidden="true">›</span>
    </li>
    <li class="ethds-breadcrumb__item">
      <a href="/services" class="ethds-breadcrumb__link">Services</a>
      <span class="ethds-breadcrumb__separator" aria-hidden="true">›</span>
    </li>
    <li class="ethds-breadcrumb__item">
      <span aria-current="page" class="ethds-breadcrumb__current">Renew licence</span>
    </li>
  </ol>
</nav>
```

```css
.ethds-breadcrumb { font-size: var(--ethds-text-sm); }
.ethds-breadcrumb__list {
  display: flex; flex-wrap: wrap; align-items: center; gap: var(--ethds-space-1);
  margin: 0; padding: 0; list-style: none;
}
.ethds-breadcrumb__item { display: inline-flex; align-items: center; gap: var(--ethds-space-1); }
.ethds-breadcrumb__link { color: var(--ethds-color-interactive); text-decoration: underline; }
.ethds-breadcrumb__link:focus-visible { outline: 2px solid var(--ethds-color-focus); outline-offset: 2px; }
.ethds-breadcrumb__current { color: var(--ethds-color-text-secondary); }
.ethds-breadcrumb__separator { color: var(--ethds-color-text-disabled); }
```

Static, server-rendered markup — no JavaScript needed. Only the last
item omits the `<a>` and carries `aria-current="page"`; every separator
is `aria-hidden` since the visual `›` is decorative.

## Props

| Prop | Type | Notes |
|---|---|---|
| `items` | `{ label, href? }[]` | last item = current page |
| `label` | `string` | landmark name, default "Breadcrumb" |
