# Link

Use a link to take citizens to another page. Link text describes the
destination, so it makes sense read on its own.

<Example>
  <Text>
    Read the <Link href="#">full guidance</Link> before you apply.
  </Text>
</Example>

```tsx
import { Link } from '@ethds/react';

<Link href="/guidance">full guidance</Link>
```

## When to use this component

Use a link for **navigation** — going to another page or resource. Use
descriptive text ("track your application"), never "click here".

## When not to use this component

Don't use a link styled as an action — if it *does* something (submits,
saves, deletes), use a [Button](button).

## Variants

<Example>
  <div style={{ display: 'flex', gap: '2rem', flexWrap: 'wrap' }}>
    <Link href="#">Default link</Link>
    <Link href="#" variant="subtle">Subtle link</Link>
    <Link href="https://example.gov.et" external>External link</Link>
  </div>
</Example>

- **external** sets `target="_blank"` with safe
  `rel="noopener noreferrer"`. Add a visually-hidden "(opens in new tab)"
  note in the link text for screen-reader users.

## Accessibility

- Renders a native `<a href>` — real links work with the keyboard,
  screen readers, and "open in new tab" for free.
- Underlined by default so links don't rely on colour alone.
- Link colour `--ethds-color-interactive` (6.40:1 on white); visible
  focus ring.
- Pass translated text as children — the text is the accessible name.

## Plain HTML

```html
<a href="/guidance" class="ethds-link">full guidance</a>

<a href="/guidance" class="ethds-link ethds-link--subtle">Subtle link</a>

<a href="https://example.gov.et" class="ethds-link" target="_blank" rel="noopener noreferrer">
  External link
  <span class="ethds-visually-hidden">(opens in new tab)</span>
</a>
```

```css
.ethds-link {
  color: var(--ethds-color-interactive);
  text-decoration: underline;
  text-underline-offset: 2px;
}
.ethds-link:hover { color: var(--ethds-color-interactive-hover); }
.ethds-link:focus-visible { outline: 2px solid var(--ethds-color-focus); outline-offset: 2px; }
.ethds-link--subtle { color: var(--ethds-color-text); text-decoration: none; }
.ethds-link--subtle:hover { text-decoration: underline; }
```

A plain `<a>` — no JavaScript needed. `target="_blank"` always pairs
with `rel="noopener noreferrer"` (never one without the other), and the
visually-hidden note ensures screen-reader users hear that the link
opens a new tab, since the visual icon convention some sites use for
this doesn't reach them.

## Props

| Prop | Type | Default |
|---|---|---|
| `variant` | `'default' \| 'subtle'` | `'default'` |
| `external` | `boolean` | `false` |
| …plus all native `<a>` props | | |
