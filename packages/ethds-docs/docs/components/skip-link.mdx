# Skip link

Lets keyboard users bypass the repeated header and navigation and jump
straight to the main content. Visually hidden until focused.

<Example>
  <div style={{ position: 'relative', minHeight: 60 }}>
    <SkipLink href="#example-main">Skip to main content</SkipLink>
    <p style={{ margin: 0 }}>
      Click inside this box, then press <kbd>Tab</kbd> — the skip link
      appears when it receives focus.
    </p>
  </div>
</Example>

```tsx
import { SkipLink } from '@ethds/react';

// First element inside <body>, pointing at your main landmark:
<SkipLink href="#main">Skip to main content</SkipLink>
…
<main id="main">…</main>
```

## When to use this component

On **every page**, as the very first focusable element — every ETHDS
template assumes it. Without it, a keyboard user tabs through the whole
header and nav on every single page.

## When not to use this component

There's no page that shouldn't have one.

## Accessibility

- A real `<a href="#main">` — activating it moves reading/focus position
  to the main landmark.
- Visually hidden off-screen until `:focus`, then shown prominently on
  the Star Blue banner with a high-contrast focus outline.
- Pass the text translated ("Skip to main content" in the active
  language).

## Plain HTML

The very first thing inside `<body>`, on every page.

```html
<a href="#main" class="ethds-skip-link">Skip to main content</a>
…
<main id="main">…</main>
```

```css
.ethds-skip-link {
  position: absolute;
  left: -9999px;
  top: 0;
  z-index: 100;
  padding: var(--ethds-space-2) var(--ethds-space-4);
  background: var(--ethds-color-primary);
  color: var(--ethds-white);
  text-decoration: underline;
  border-radius: 0 0 var(--ethds-radius-md) 0;
}
.ethds-skip-link:focus {
  left: 0;
  outline: 2px solid var(--ethds-yellow-400);
  outline-offset: 2px;
}
```

No JavaScript — a plain in-page anchor link, pushed off-screen until it
receives keyboard focus. This is the exact markup already used at the
top of the [404 page](/docs/templates/404-page#plain-html) template.

## Props

| Prop | Type | Default |
|---|---|---|
| `href` | `string` | `'#main'` |
| `children` | `ReactNode` | required — translated label |
