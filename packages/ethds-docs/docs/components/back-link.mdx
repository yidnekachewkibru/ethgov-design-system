# Back link

A single-step link back to the previous page in a service flow —
distinct from the browser's own back button, which can leave a service.

<Example>
  <BackLink href="#">Back</BackLink>
</Example>

```tsx
import { BackLink } from '@ethds/react';

<BackLink href="/application/step-2">Back</BackLink>
```

## When to use this component

At the top of every step in a multi-step service (alongside a
[Step indicator](step-indicator.mdx)) so the citizen can return to the
previous step without losing entered data.

## When not to use this component

Don't use it as the only way to navigate — it complements, not replaces,
a browser back button and a [Breadcrumb](breadcrumb.mdx) for hierarchy.
Don't use it on the first step of a flow, where there is nowhere to go
back to.

## Accessibility

- A native `<a>` — keyboard-operable and screen-reader-friendly for free.
- The chevron is decorative (`aria-hidden`); the link's accessible name
  comes entirely from its text ("Back"), which is translatable.

## Plain HTML

```html
<a href="/application/step-2" class="ethds-back-link">
  <svg class="ethds-icon ethds-icon--sm" viewBox="0 0 24 24" fill="none" stroke="currentColor"
       stroke-width="2" stroke-linecap="round" stroke-linejoin="round" aria-hidden="true" focusable="false">
    <path d="m15 18-6-6 6-6" />
  </svg>
  Back
</a>
```

```css
.ethds-back-link {
  display: inline-flex;
  align-items: center;
  gap: var(--ethds-space-1);
  font-weight: var(--ethds-font-medium);
  color: var(--ethds-color-interactive);
  text-decoration: underline;
}
.ethds-back-link:hover { color: var(--ethds-color-interactive-hover); }
.ethds-back-link:focus-visible { outline: 2px solid var(--ethds-color-focus); outline-offset: 2px; }
```

A plain `<a href>` to the previous step's URL — no JavaScript, and no
`history.back()`, since that can leave the service entirely if the
citizen arrived from an external link.

## Props

| Prop | Type | Notes |
|---|---|---|
| `href` | `string` | required — the previous step's URL |
| `children` | `ReactNode` | required — the link text (translatable) |
