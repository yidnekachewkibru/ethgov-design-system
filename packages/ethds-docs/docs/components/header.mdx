# Header

The government identity and primary navigation — the consistent banner
that tells a citizen this is an official service.

<Example>

## When to use this component

On **every page** of every ETHDS service. The shared identity — Star Blue
banner, emblem slot, consistent navigation — is how citizens recognise
official services and spot impersonation.

## When not to use this component

Don't rebuild a per-ministry header with its own colours or structure —
the delivering body is named *within* the shared identity, per the
government branding standards.

## Accessibility

- A `<header>` banner landmark; the identity is a home link with an
  explicit accessible name (`homeLabel`).
- The primary nav is a labelled `<nav>`; the current item has
  `aria-current="page"`.
- White-on-Star-Blue text is 8.31:1; focus rings are visible against the
  blue.
- Put the [Language switcher](language-switcher) in `actions` so it's on
  every page, early in the tab order.

## Props

| Prop | Type | Notes |
|---|---|---|
| `serviceName` | `ReactNode` | required, translatable |
| `homeHref` / `homeLabel` | `string` | identity link + its accessible name |
| `logo` | `ReactNode` | emblem slot |
| `nav` | `{ label, href, current? }[]` | primary navigation |
| `navLabel` | `string` | nav landmark name |
| `actions` | `ReactNode` | e.g. LanguageSwitcher, account |
