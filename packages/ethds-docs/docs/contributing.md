# Contributing

ETHDS is open source and developed in the open. Contributions from
designers, developers, accessibility specialists, translators, and
government service teams are all welcome.

- **Start here:** the
  [CONTRIBUTING guide](https://github.com/yidnekachewkibru/ethgov-design-system/blob/main/CONTRIBUTING.md)
- **Be respectful:** the
  [Code of Conduct](https://github.com/yidnekachewkibru/ethgov-design-system/blob/main/CODE_OF_CONDUCT.md)
  governs participation.
- **Significant changes** (new components, patterns, breaking token
  changes) go through an RFC — see
  [the RFC process](https://github.com/yidnekachewkibru/ethgov-design-system/tree/main/packages/ethds-rfcs).

## Translations

This site supports six languages. English is authored; the other five
fall back to English until translated. Translating the documentation is a
high-value contribution — the localization workflow is defined in the
[Localization Framework](https://github.com/yidnekachewkibru/ethgov-design-system/blob/main/docs/phases/phase-6-localization-framework.md)
(Phase 6).

## Accessibility & localization are merge requirements

Any change that adds or alters UI must show how WCAG 2.2 AA and the six
supported languages were considered — this is a requirement for merging,
not optional review feedback.
